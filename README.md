# 🎨 EasySticker - AI 貼圖製作工具

免費的 AI 貼圖製作工具，支援 LINE/WeChat/Telegram 多平台，一鍵完成貼圖裁切、去背、匯出。

## ✨ 功能特色

- **自動裁切**：支援 4×3 / 6×4 / 8×6 等多種網格佈局，精準分離每張貼圖
- **AI 去背**：自動偵測並移除白色/淺色背景，保留邊緣平滑過渡
- **多平台支援**：LINE (370×320)、WeChat (240×240)、Telegram (512×512)、WhatsApp (512×512)、Discord (128×128)
- **用語編輯**：每張貼圖可自訂文字內容
- **ZIP 匯出**：一鍵下載所有貼圖為 ZIP 檔案
- **完全本地運行**：所有處理都在瀏覽器中完成，圖片不會上傳到伺服器

## 🚀 使用方式

1. 直接打開 `index.html` 檔案
2. 上傳貼圖紙板圖片
3. 選擇貼圖數量（12/24/36/48 張）
4. 選擇輸出平台
5. （可選）編輯每張貼圖的文字
6. 點擊「開始製作貼圖」
7. 下載結果
